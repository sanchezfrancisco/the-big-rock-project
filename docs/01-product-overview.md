# Product Overview

Semantic Code Navigator is a local-first retrieval system for private Python repositories.

Core capabilities:
- Index Python source code into local SQLite.
- Ask technical questions and return verifiable file/line citations.
- Explain ranking with vector and keyword score components.
- Evaluate retrieval quality with recall and latency metrics.
- Tune hybrid retrieval weight and track quality trends.
- Serve a local web UI for operators and developers.

Primary surfaces:
- CLI (`scn ...`)
- HTTP API (FastAPI)
- Local Web UI (`/`)
