# Web UI Guide

URL:
- `http://127.0.0.1:8000` (or selected port)

Panels:
- Runtime: index path, backend selector, hybrid weight, dataset path, top-k, weights, cache size.
- Index: repository indexing.
- Ask & Explain: question answering and ranking explainability.
- Eval & Tune: quality measurement and tuning.
- Cache: stats and pruning.
- Status: current index/cache state.
- Observability: recent event log.

Autocomplete behavior:
- Suggestions update while typing.
- `Tab` accepts active suggestion.
- Suggestions adapt to selected backend and current query text.
